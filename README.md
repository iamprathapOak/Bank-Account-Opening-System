# Bank-Account-Opening-System

This is a simple Java-based application that demonstrates the concept of  **Method Overloading** by handling different seat of bank account information.

## project Overview
  The project allows displaying user account detils such as Bank Name, Branch, Account Number, and Balance. It used multiple 'main' method (Method Overloading) to show how Java handle different entry points or parameters within same class structure.

  ## Key Java Concepts Used
  * **Method Overloading:** multilple 'main' method with different parameter lists.
  * **Ststic Method:** To access methods without creating objects.
  * **String Formatting:** Displaying account detils like Balance, Mobile Number, and Address.
